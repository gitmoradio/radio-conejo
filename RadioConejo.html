Le Code d'Accès

Comment entrer · Comment changer
1
Quand tu ouvres le site, tu vois l'écran de verrouillage. Choisis d'abord ton concert dans la liste, puis tape le mot de passe et appuie sur ENTRAR.
2
Le code par défaut est conejo (lapin en espagnol 🐰). Simple!
3
Pour changer le code, ouvre RadioConejo.html dans un éditeur de texte (Bloc-notes, TextEdit, VS Code…). Cherche cette ligne en haut du script:
const ACCESS_CODE = 'conejo';
Remplace conejo par ton propre mot. Ex: dbmf2026.
💡 Bon à savoir — une fois connecté, le site reste déverrouillé pendant toute la session du navigateur. Le concert choisi est aussi mémorisé.
⚠️ Attention — ce code est visible dans le fichier HTML. C'est fait pour bloquer les curieux, pas pour protéger des secrets d'état 😄
🗓️

Changer de Concert

Sélecteur intégré · Ou modifier le code
1
Depuis l'écran de verrouillage — utilise les boutons de sélection de concert. Clique sur une ville pour changer le countdown, les couleurs et les infos du show. C'est tout!
2
Ajouter un nouveau show — ouvre le fichier dans un éditeur et cherche le tableau DBMF_SHOWS. Chaque ligne = un show:
{ city: 'Marseille', venue: 'Orange Vélodrome', date: '2026-07-01T21:00:00', flag: '🇫🇷', cityColor: '#009EE0' },
Copie une ligne, colle-la, et change les valeurs.
3
cityColor est le code couleur hexadécimal de la ville ou du club local (ex: OM blue pour Marseille). Cherche "couleur principale [nom club/ville]" sur Google.
4
Show par défaut — change la variable DEFAULT_SHOW_INDEX (0 = premier show de la liste) pour choisir quel show s'affiche au premier chargement.
💡 Forker pour un autre pote — copie RadioConejo.html, renomme-le (ex: RadioConejoLondres.html), change DEFAULT_SHOW_INDEX et pousse-le sur un autre repo GitHub.
🎬

Swapper les Vidéos

YouTube IDs · Copier-coller
1
Cherche le tableau VIDEO_PANES dans le script. Chaque ligne = un panneau vidéo.
2
Pour trouver l'ID YouTube d'une vidéo: ouvre la vidéo sur YouTube. Regarde l'URL — la partie après ?v= est l'ID. Ex: dans youtube.com/watch?v=HQnC1UHBvWA, l'ID est HQnC1UHBvWA.
3
Remplace l'ID dans le tableau. Les IDs terminant par _YT sont des placeholders — remplace-les pour que la vidéo s'affiche.
{ label: '🎬 Mon titre', youtubeId: 'HQnC1UHBvWA' },
4
Les boutons 1 / 2 / 4 changent la grille d'affichage. La grille 4 panneaux utilise les 4 premières entrées du tableau.
💡 Track du jour — tu peux aussi mettre à jour les IDs YouTube dans le tableau DAILY_TRACKS de la même façon.
🚀

Mettre en Ligne

GitHub Pages · Gratuit · 5 minutes
1
Crée un compte sur github.com si tu n'en as pas (c'est gratuit).
2
Clique New repository (bouton vert). Nomme-le radio-conejo. Coche Public. Clique Create repository.
3
Clique Add file → Upload files. Glisse RadioConejo.html et index.html. Clique Commit changes.
4
Va dans Settings → Pages. Sous Source, choisis Deploy from a branch → main → / (root). Clique Save.
5
Attends 2 minutes, rafraîchis. Ton URL: https://[username].github.io/radio-conejo/ — partage! 🐸
💡 Mettre à jour — retourne dans ton repo GitHub, clique le fichier puis le crayon ✏️ pour éditer directement dans le navigateur. Sauvegarde et c'est en ligne instantanément.
⚠️ Le site est public — n'importe qui avec le lien voit l'écran de connexion, mais pas le contenu sans le code.
