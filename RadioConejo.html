🐸 GUIDE

🔐 Code d'accès

Modifie ACCESS_CODE en haut du script. Par défaut: conejo. Stocké en sessionStorage.

🗓️ Date du concert

Modifie CONCERT_DATE si nécessaire. Format: 'YYYY-MM-DDTHH:MM:SS'

🐸 Membres du crew

Modifie le tableau CREW. Chaque entrée: { name, nickname, emoji, organizer? }. Une ligne = un membre.

🎵 Tracks du jour

Modifie DAILY_TRACKS. Chaque entrée: { title, album, note, youtubeId }. Tourne par jour de l'année.

🎬 Panneaux vidéo

Modifie VIDEO_PANES. Chaque entrée: { label, youtubeId }. Les IDs marqués _YT sont des placeholders — remplace-les par de vrais IDs YouTube.

📰 News automatiques

Récupère le flux Google News RSS "Bad Bunny" via l'API rss2json (sans clé). Fonctionne offline avec un message de fallback.

📻 Radio SoundCloud

Pointe sur le profil officiel Bad Bunny. Pour changer: remplace l'URL dans le src de l'iframe player.

🚀 Déploiement GitHub Pages

  • Push RadioConejo.html + index.html dans un repo public
  • Settings → Pages → Source: main / root
  • URL: https://[ton-user].github.io/[repo]/
  • Partage le lien avec le crew!
