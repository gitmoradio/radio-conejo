Le Code d'Accès

Comment entrer · Comment changer
1
Quand tu ouvres le site, tu vois un écran de verrouillage. Tape le mot de passe dans la case et appuie sur ENTRAR.
2
Le code par défaut est conejo (lapin en espagnol 🐰). Simple, non ?
3
Pour changer le code, ouvre RadioConejo.html dans un éditeur de texte (Bloc-notes, TextEdit, VS Code…). Cherche cette ligne en haut du script:
const ACCESS_CODE = 'conejo';
Remplace conejo par n'importe quel mot. Ex: dbmf2025.
💡 Bon à savoir — une fois connecté, le site reste déverrouillé pendant toute la session (jusqu'à la fermeture de l'onglet). Pas besoin de retaper à chaque fois.
⚠️ Attention — ce code est visible dans le fichier HTML. C'est fait pour bloquer les curieux, pas pour protéger des secrets d'état 😄
🐸

Ajouter / Retirer du Crew

Chaque membre = une ligne de code
1
Ouvre RadioConejo.html dans un éditeur de texte. Cherche le tableau CREW dans le script — ça ressemble à ça:
const CREW = [ { name: 'Norman', nickname: 'El Organizador', emoji: '🐰', organizer: true }, { name: 'Sofia', nickname: 'La Visionaria', emoji: '🌟' }, // ... etc ];
2
Ajouter quelqu'un — copie une ligne existante, colle-la en dessous, et change le nom, le surnom et l'emoji. N'oublie pas la virgule à la fin de chaque ligne (sauf la dernière).
3
Retirer quelqu'un — efface sa ligne entière. Vérifie que la ligne précédente se termine par une virgule.
4
Le champ organizer: true donne un badge spécial 👑 — parfait pour l'organisateur. Il n'y en a qu'un normalement!
💡 Trouver un emoji — sur Mac: appuie sur Ctrl + Cmd + Espace. Sur Windows: appuie sur Win + . pour ouvrir le clavier emoji.
🗓️

Changer de Concert

Une autre ville · Une autre date
1
Cherche le bloc CONCERT tout en haut du script. Il ressemble à ça:
const CONCERT = { artist: 'Bad Bunny', tour: 'DeBÍ TiRAR MáS FOToS', city: 'Marseille', venue: 'Orange Vélodrome', date: '2025-07-01T21:00:00', flag: '🇫🇷', cityColor: '#009EE0', };
2
Change city, venue, date, flag et cityColor pour ton concert. Le format de la date: ANNÉE-MOIS-JOURTHHEURE:MIN:SEC — par exemple 2025-08-15T20:30:00.
3
cityColor est la couleur principale de ta ville ou de la salle. C'est un code hexadécimal (le # suivi de 6 chiffres/lettres). Cherche "couleur principale [nom ville]" sur Google pour en trouver une.

Swapper les Vidéos

YouTube IDs · Copier-coller
1
Cherche le tableau VIDEO_PANES dans le script. Chaque ligne = un panneau vidéo.
2
Pour trouver l'ID YouTube d'une vidéo: ouvre la vidéo sur YouTube. Regarde l'URL — la partie après ?v= est l'ID. Par exemple dans youtube.com/watch?v=HQnC1UHBvWA, l'ID est HQnC1UHBvWA.
3
Remplace l'ID dans le tableau. Les IDs qui se terminent par _YT sont des placeholders — ils doivent absolument être remplacés pour que la vidéo s'affiche.
{ label: '🎬 Mon titre', youtubeId: 'HQnC1UHBvWA' },
4
La grille affiche 1, 2 ou 4 vidéos selon le bouton sélectionné. Les 4 premières entrées du tableau sont utilisées pour la grille 4 panneaux.
💡 Track du jour — tu peux aussi mettre à jour les IDs dans le tableau DAILY_TRACKS de la même façon. Chaque track a un champ youtubeId.
🚀

Mettre en Ligne

GitHub Pages · Gratuit · 5 minutes
1
Crée un compte sur github.com si tu n'en as pas (c'est gratuit).
2
Clique sur New repository (le bouton vert). Donne-lui un nom, ex: radio-conejo. Coche Public. Clique Create repository.
3
Dans ton nouveau repo, clique Add file → Upload files. Glisse RadioConejo.html et index.html dans la zone de dépôt. Clique Commit changes.
4
Va dans Settings → Pages (dans le menu de gauche). Sous Source, choisis Deploy from a branch, sélectionne main et / (root). Clique Save.
5
Attends 1-2 minutes, rafraîchis la page. Ton URL sera:
https://[ton-username].github.io/radio-conejo/
Partage-la avec le crew! 🐸
💡 Mettre à jour — pour changer quelque chose après la mise en ligne, retourne dans ton repo GitHub, clique sur le fichier, puis sur le crayon ✏️ pour l'éditer directement dans le navigateur. Sauvegarde et le site se met à jour automatiquement.
⚠️ Le site est public — n'importe qui avec le lien peut voir la page de connexion (mais pas entrer sans le code). Si tu veux garder le lien discret, évite de le poster sur des réseaux publics.
